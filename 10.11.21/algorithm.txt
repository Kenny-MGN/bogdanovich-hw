01. Инициализуем/создаем репозиторий.
02. Создаем файл index.html и индексируем его командой git add index.html.
03. Делаем снимок командой git commit -m "initial".
04. Создаем побочную ветку и переключаемся на неё командой git checkout -b slave.
05. Изменяем строку в файле, индексируем его командой git add.
6. Делаем коммит git commit -m "modified file in slave tree".
07. Переключаемся обратно на ветку master команой git checkout master.
08. Изменяем ту же самую строку, индексируем файл и делаем коммит.
09. Проводим слияние веток командой git merge slave.
10. Редактируем файл в текстовом редакторе, выбираем вариант строки/кода, удаляем служебные символы.
11. Индексируем файл и делаем коммит командой git commit.
12. Команда git commit без флага -m вызывает текстовый редактор с описанием описание возникшей ситуации.
13. Сохраняем комментарий. Конфликт решен.
